---
title: List Authors
api: GET /authors
---

Retrieves a paginated list of all authors in the library system.

## Authentication

This endpoint requires HTTP Basic Authentication.

## Query Parameters

<ParamField query="page" type="integer" default="0">
  The page number to retrieve (zero-based)
</ParamField>

<ParamField query="size" type="integer" default="20">
  The number of items per page
</ParamField>

<ParamField query="sort" type="string">
  Sort criteria in the format: property(,asc|desc). Default sort order is ascending. Multiple sort criteria are supported.
  
  Example: `name,asc` or `age,desc`
</ParamField>

## Response

The response is a paginated object containing an array of authors.

<ResponseField name="content" type="array">
  Array of author objects
  
  <ResponseField name="id" type="long">
    The unique identifier of the author
  </ResponseField>
  
  <ResponseField name="name" type="string">
    The name of the author
  </ResponseField>
  
  <ResponseField name="age" type="integer">
    The age of the author
  </ResponseField>
</ResponseField>

<ResponseField name="pageable" type="object">
  Pagination information
  
  <ResponseField name="pageNumber" type="integer">
    Current page number (zero-based)
  </ResponseField>
  
  <ResponseField name="pageSize" type="integer">
    Number of items per page
  </ResponseField>
  
  <ResponseField name="offset" type="long">
    Offset of the current page
  </ResponseField>
</ResponseField>

<ResponseField name="totalPages" type="integer">
  Total number of pages
</ResponseField>

<ResponseField name="totalElements" type="long">
  Total number of authors
</ResponseField>

<ResponseField name="last" type="boolean">
  Whether this is the last page
</ResponseField>

<ResponseField name="first" type="boolean">
  Whether this is the first page
</ResponseField>

<ResponseField name="size" type="integer">
  Number of items in the current page
</ResponseField>

<ResponseField name="number" type="integer">
  Current page number (zero-based)
</ResponseField>

<ResponseField name="numberOfElements" type="integer">
  Number of elements in the current page
</ResponseField>

<ResponseField name="empty" type="boolean">
  Whether the page is empty
</ResponseField>

## Status Codes

<ResponseField name="200" type="OK">
  Authors successfully retrieved
</ResponseField>

<ResponseField name="401" type="Unauthorized">
  Authentication credentials are missing or invalid
</ResponseField>

## Example Request

```bash
curl -X GET "http://localhost:8080/authors?page=0&size=10&sort=name,asc" \
  -u username:password
```
