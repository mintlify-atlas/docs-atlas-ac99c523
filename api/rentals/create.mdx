---
title: "Create Rental"
api: "POST /books/{isbn}/user/{user_id}"
description: "Create a new book rental for a specific user. This endpoint automatically decrements the book's stock by 1 and sets a 2-week loan period."
---

## Authentication

This endpoint requires HTTP Basic Authentication.

## Path Parameters

<ParamField path="isbn" type="string" required>
  The ISBN of the book to rent
</ParamField>

<ParamField path="user_id" type="integer" required>
  The ID of the user renting the book
</ParamField>

## Response

<ResponseField name="id" type="integer">
  Unique identifier for the rental
</ResponseField>

<ResponseField name="user" type="object">
  User who rented the book
  <ResponseField name="id" type="integer">
    User ID
  </ResponseField>
  <ResponseField name="firstname" type="string">
    User's first name
  </ResponseField>
  <ResponseField name="lastname" type="string">
    User's last name
  </ResponseField>
  <ResponseField name="email" type="string">
    User's email address
  </ResponseField>
  <ResponseField name="displayPicture" type="string" optional>
    URL to user's display picture
  </ResponseField>
</ResponseField>

<ResponseField name="book" type="object">
  Book being rented
  <ResponseField name="isbn" type="string">
    Book ISBN
  </ResponseField>
  <ResponseField name="title" type="string">
    Book title
  </ResponseField>
  <ResponseField name="stock" type="integer">
    Current stock count (decremented by 1 after rental)
  </ResponseField>
  <ResponseField name="coverImage" type="string" optional>
    URL to book cover image
  </ResponseField>
  <ResponseField name="author" type="object">
    Book author details
    <ResponseField name="id" type="integer">
      Author ID
    </ResponseField>
    <ResponseField name="name" type="string">
      Author name
    </ResponseField>
    <ResponseField name="age" type="integer">
      Author age
    </ResponseField>
  </ResponseField>
</ResponseField>

<ResponseField name="loanDate" type="string">
  ISO 8601 timestamp when the book was rented (UTC)
</ResponseField>

<ResponseField name="returnDate" type="string">
  ISO 8601 timestamp when the book is due (2 weeks from loan date)
</ResponseField>

<ResponseField name="returned" type="boolean">
  Whether the book has been returned (always false for new rentals)
</ResponseField>

## Error Responses

<ResponseField name="404" type="error">
  Book with the specified ISBN or user with the specified ID not found
</ResponseField>

<ResponseField name="400" type="error">
  Book is out of stock (stock = 0)
</ResponseField>

## Automatic Stock Management

When a rental is created, the book's stock is automatically decremented by 1. If the book has no stock available (stock = 0), the request will fail with a 400 Bad Request error.

## Loan Period

All rentals have a 2-week loan period. The `returnDate` is automatically set to 2 weeks (14 days) from the `loanDate`.
