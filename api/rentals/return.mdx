---
title: "Return Book"
api: "PUT /rentals/{rental_id}/return"
description: "Mark a rental as returned and automatically increment the book's stock by 1."
---

## Authentication

This endpoint requires HTTP Basic Authentication.

## Path Parameters

<ParamField path="rental_id" type="integer" required>
  The unique identifier of the rental to mark as returned
</ParamField>

## Response

<ResponseField name="id" type="integer">
  Unique identifier for the rental
</ResponseField>

<ResponseField name="user" type="object">
  User who rented the book
  <ResponseField name="id" type="integer">
    User ID
  </ResponseField>
  <ResponseField name="firstname" type="string">
    User's first name
  </ResponseField>
  <ResponseField name="lastname" type="string">
    User's last name
  </ResponseField>
  <ResponseField name="email" type="string">
    User's email address
  </ResponseField>
  <ResponseField name="displayPicture" type="string" optional>
    URL to user's display picture
  </ResponseField>
</ResponseField>

<ResponseField name="book" type="object">
  Book that was returned
  <ResponseField name="isbn" type="string">
    Book ISBN
  </ResponseField>
  <ResponseField name="title" type="string">
    Book title
  </ResponseField>
  <ResponseField name="stock" type="integer">
    Current stock count (incremented by 1 after return)
  </ResponseField>
  <ResponseField name="coverImage" type="string" optional>
    URL to book cover image
  </ResponseField>
  <ResponseField name="author" type="object">
    Book author details
    <ResponseField name="id" type="integer">
      Author ID
    </ResponseField>
    <ResponseField name="name" type="string">
      Author name
    </ResponseField>
    <ResponseField name="age" type="integer">
      Author age
    </ResponseField>
  </ResponseField>
</ResponseField>

<ResponseField name="loanDate" type="string">
  ISO 8601 timestamp when the book was originally rented
</ResponseField>

<ResponseField name="returnDate" type="string">
  ISO 8601 timestamp when the book was due
</ResponseField>

<ResponseField name="returned" type="boolean">
  Whether the book has been returned (always true after this operation)
</ResponseField>

## Error Responses

<ResponseField name="404" type="error">
  Rental with the specified ID not found
</ResponseField>

## Automatic Stock Management

When a book is returned, the associated book's stock is automatically incremented by 1. This makes the book available for other users to rent.

<RequestExample>

```bash cURL
curl -X PUT https://api.example.com/rentals/123/return \
  -u "username:password" \
  -H "Content-Type: application/json"
```

</RequestExample>
